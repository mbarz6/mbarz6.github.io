(Math note: It turns out that in any finite field, all functions are realizable as polynomials! There are a few ways of noticing this: First, Lagrange interpolation works, meaning that there's a degree q polynomial realizing any function. Additionally, one can do it by first noticing that the function which sends everything to 0 except for 0, which is sent to -1, is a polynomial of degree -1, namely xq-1 - 1, essentially by Fermat's little theorem for finite fields. Then, every possible function can be represented as a sum combination of functions of the form a(x-b)q-1 - 1/)
