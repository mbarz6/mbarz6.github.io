(Math note: It turns out that in any finite field, all functions are realizable as polynomials! One can do it by first noticing that the function which sends everything to 0 except for 0, which is sent to -1, is a polynomial of degree -1, namely xq-1 - 1, essentially by Fermat's little theorem for finite fields. Then, every possible function can be represented as a sum combination of functions of the form a(x-b)q-1 - 1. Additionally, without knowing who Fermat was, one can discover this simply by cleverly exploiting finiteness of the field: Take the product of x-a over all but one element of the field to get a function which is zero almost everywhere, and form linear combinations of these functions to finish.)
