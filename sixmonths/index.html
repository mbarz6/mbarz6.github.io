However, since F8 is finite, there are only finitely many functions from F8 to itself, in particular only 88. Of those finitely many functions, how many are polynomials? Phrased differently, if instead of considering two polynomials to be the same if they have exactly the same coefficients and instead say two polynomials are the same if the map the same inputs to the same outputs, how many distinct polynomials are there?
