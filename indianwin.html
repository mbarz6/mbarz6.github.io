Math note: It turns out that every algebraic number of degree n is associated with an n-by-n matrix of rational coefficients which obeys the same polynomials as the algebraic number. If the number is an algebraic integer, the matrix can be guaranteed to have integral coefficients. One way of thinking about this result is as an explicit construction performed with the Cayley-Hamiliton Theorem. Another way of viewing it is to notice that if the algebraic number is a, then the smallest subfield of the complex numbers containing the rational numbers and a is a vector space over the rational numbers of a certain (finite) dimension. Then, we can view multiplication by a to be a linear transformation of this vector space, and the associated matrix has the desired properties.
